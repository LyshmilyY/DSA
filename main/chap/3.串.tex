\subsection{串的定义}

\begin{\definition}[串的定义]
    串($String$)是由零个或者多个字符组成的有限序列
    $$S = ''a_{1}a_{2}\cdots a_{n}'' (n\geq 0)$$
    \begin{itemize}
        \item $S$ 是串名, 双引号内的字符序列是串的值；$a_{i}(1\leq i \leq n)$ 可以是字母、数字或其他字符
        \item 串中字符的数目 $n$ 称为串的长度, 零个字符的串称为空串, 长度为零
        \item 串中任意个连续字符组成的子序列称为该串的子串, 包含子串的串称为主串
        \item 字符在系列中的序号为该字符在串中的位置, 子串在主串中的位置以子串的第一个字符在主串中的位置表示
        \item 两个串相等当且仅当两个串长度相等且各个位置的字符都对应相等, 由一个或多个空格组成的串称空格串, 空串用符号 $\emptyset$ 表示
    \end{itemize}
\end{definition}
